📊 랜드코어 사용가이드

랜드코어 부동산 투자 분석 도구 사용법을 알아보세요

🚀 시작하기

랜드코어 분석기는 네이버 부동산에서 아파트/빌라/상가 매물을 분석하는 크롬 확장 프로그램입니다.

  1. 랜드코어 카카오 로그인을 하면 네이버 부동산 상가 페이지로 이동합니다.
  2. 원하는 지역으로 지도를 이동하고, 필터(아파트/빌라/상가/점포, 매매/월세)를 설정합니다.
  3. 지도상 동그란 매물을 클릭하면 매물목록(자동)이 스크롤되면서 화면에 표시합니다.
  4. 지도상에 동그란 원매물을 클릭하면 사이드패널에서 자동으로 인식되어 실시간 분석이 시작됩니다!

💡 팁: 스크롤은 자동으로 이루어지며 많은 매물이 분석됩니다(로딩하면서 분석). 실시간으로 사이드패널에 결과가 업데이트됩니다!

✨ 주요 기능

📈 실시간 분석

지도를 이동하면 자동으로 매물을 분석합니다. 버튼을 누를 필요 없이 실시간 업데이트!

📊 평당가 계산

매매/월세 매물의 평당가를 자동 계산하여 층별로 정리해 보여줍니다.

🏷️ 배지 표시

매물 목록에 평당가와 평수 배지가 자동으로 표시되어 한눈에 비교 가능!

💰 수익률 계산기

매물 상세정보를 자동으로 불러와 수익률을 계산합니다.

📊 배후분석

분석하고자 하는 법정동에 대한 배후(세대수/인구/직장등)를 분석합니다.

📍 최저가 매물

평당가 기준 가장 저렴한 매매 매물 정보를 한눈에 확인!

💵 수익률 계산기 사용법

  1. 네이버 부동산에서 원하는 매물을 클릭하여 상세정보 팝업을 엽니다.
  2. 사이드패널의 📊 수익률 계산기 버튼을 클릭합니다.
  3. 매매가, 면적, 보증금/월세 정보가 자동으로 입력됩니다.
  4. 추가 비용(취득세, 대출 등)을 입력하고 수익률을 확인하세요!

💡 팁: 매물 상세정보 팝업이 열린 상태에서 계산기 버튼을 눌러야 정보가 자동으로 채워집니다.

📄 저장하기

수익률 계산 결과를 이미지 저장하여 보관하거나 공유할 수 있습니다:

  1. 수익률 계산기에서 💾 저장 버튼을 클릭합니다.
  2. 저장 버튼을 클릭하면 법정동이름으로 이미지 파일이 다운로드됩니다!

💡 팁: 세로 A4 2페이지로 출력됩니다 (1페이지: 분석 결과, 2페이지: 현금흐름표)

📖 분석 결과 이해하기

층별 평당가 표

매매와 월세 매물의 평당가를 1층, 2층, 상층(3층 이상), 지하로 구분하여 보여줍니다. 숫자는 절사평균(상하위 10% 제외)으로 계산됩니다.

평수 분포 차트

월세 매물의 평수 분포를 10평 단위로 시각화합니다. 어떤 크기의 매물이 많은지 한눈에 파악할 수 있습니다.

부동산 TOP 3

가장 많은 매물을 보유한 중개사 3곳을 보여줍니다. 월세와 매매 매물 수가 각각 표시됩니다.

⚠️ 참고: "중개사 N곳" 형태의 매물은 특정 중개사로 집계되지 않습니다.

⚖️ 법적 안내

랜드코어는 다음 원칙을 준수합니다:

  • 자동 스크롤 없음 - 사용자가 직접 스크롤한 매물만 분석
  • 서버 전송 없음 - 모든 데이터는 브라우저 내에서만 처리
  • 개인정보 수집 없음 - 전화번호 등 개인정보를 수집하지 않음
  • 읽기 전용 - 네이버 부동산 페이지를 수정하지 않음 (배지 표시 제외)
