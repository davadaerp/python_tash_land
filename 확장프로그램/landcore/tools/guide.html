📊 랜드코어

부동산 데이터 통합 분석 솔루션 사용 가이드

🔐 01. 로그인 계정 인증

  1. 카카오 로그인: 상단 팝업을 통해 카카오 계정으로 간편하게 로그인합니다.
  2. 최초 사용자 인증: '구독하기' 버튼을 눌러 전화번호 본인 인증을 완료해야 합니다.
  3. 승인 요청: 개인정보 수집 동의 후 구독 요청 시, 관리자 승인 메시지가 발송되며 승인 후 서비스 이용이 가능합니다.
  4. 대시보드 활성화: 승인 완료 시 팝업 상단에 구독 정보 및 문자 충전 잔액이 표시됩니다.
  5. 마이페이지: 내 정보 관리, 요청 이력 확인, 프로그램 상세 페이지 확인이 가능합니다.

🛠️ 02. 부동산 분석 도구 통합 메뉴

로그인후 사이트패널에 분석도구 그룹에 생성되는 부동산 분석위한 핵심 버튼 메뉴입니다.

🔍 매물/실거래 검색

현재 지도 화면 기반의 실시간 매물 및 국토부 실거래가 통합 검색 기능을 제공합니다.

👥 배후분석

해당 법정동의 인구, 세대수, 주변 아파트 및 빌라 세대 정보를 정밀 분석합니다.

💰 수익률 계산기

각 구성에 맞는 수익률 분석기가 실행되며, 입력된 조건에 따라 자동 계산 및 이미지 저장을 지원합니다.

📩 문자보내기

법정동별 중개사/상담사 목록을 확인하고, 매물 정보를 알림톡이나 문자로 즉시 전송합니다.

📥 양식다운

부동산 거래에 필요한 각종 양식(기본, 법률 등)을 트리 구조로 확인하고 다운로드합니다.

🚨 NPL검색

현재 지역의 NPL 물건을 종류별(상가, 아파트 등)로 검색하며, 사건번호 복사 및 상세 지도를 제공합니다.

💡 알림: 위 모든 메뉴는 로그인 및 구독 승인이 완료된 사용자에게만 접근이 가능합니다.

📉 03. 매물 심층 분석 (분석표)

지도상의 원(매물 갯수)을 클릭하면 네이버 정보와 확장 프로그램 정보를 조합한 리스트가 출력됩니다.

[분석표 상세 기능]

  1. 데이터 조합 리스트: 네이버 기본 정보를 해치지 않는 선에서 랜드코어만의 분석 데이터를 조합하여 보여줍니다.
  2. 평형별 통계: 층별(1층, 2층, 저층, 상층)로 평형별 최소, 평균, 최대값을 분석하여 제공합니다.
  3. 실거래 & 경매 조회: 국토부 실거래 DB와 대법원 경매 데이터를 법정동 기준으로 매칭하여 리스트화합니다.
  4. 지도로 보기: 실거래 및 경매 리스트의 주소지를 기반으로 팝업 지도에 위치를 시각화합니다.
  5. 동별 통계분석: 시-군-구 내 각 동별 거래 평균 금액, 평균 년수, 금액별 거래 건수를 통계로 보여줍니다.

🚀 04. 기타 편의 기능

📊 데이터 분석

시-군-구 단위의 대규모 실거래 및 경매 목록을 한꺼번에 분석하고 통계를 도출합니다.

🔗 사이트 바로가기

인터넷등기소, 정부24 등 업무에 필요한 공공 사이트로 즉시 연결됩니다.

📁 엑셀 저장

중개사/상담사 목록을 휴대폰 멀티 문자 전송이 용이한 콤마(,) 구분 형식으로 저장합니다.

⚖️ 법적 가이드라인

[데이터 조합의 적법성]

랜드코어는 네이버 부동산의 기본 정보를 훼손하거나 수정하지 않고, 사용자의 브라우저 내에서만 부가 정보를 조합(Overlay)하여 표시합니다. 이는 사용자의 편의성을 높이는 '읽기 전용' 툴로 작동하며, 공공데이터(국토부, 대법원)를 정식 활용하므로 법적 안정성을 확보하고 있습니다.
